<h1 align="center">ChatGPT Web Speedboost Entirely</h1>

Firefox extension for reducing lag in long ChatGPT conversations.

## What It Does

- Virtualizes old offscreen message DOM.
- Restores content as you scroll.
- Reduces typing lag in large chats.
- Hides streaming paint churn until replies finish.
